# Customer-Performance-Dashboard
This project presents an interactive Customer Performance Dashboard designed to analyze customer behavior, engagement, and revenue contribution using data visualization and analytical techniques. The dashboard helps businesses understand their customers better and improve decision-making.

This project presents an interactive Customer Performance Dashboard built in Power BI using DAX measures to analyze customer behavior, engagement, and revenue contribution. The dashboard transforms raw transactional data into meaningful business insights through advanced calculations and visual analytics.

Objective
To help businesses track and evaluate:
Customer purchasing behavior Revenue contribution by customer segments Retention and engagement trends High-value vs low-value customers.
